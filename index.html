História da WEB

A história da Web se divide em duas outras histórias, a história da computação e a história da internet

Breve História da computação:

Os primeiros computadores surgiram na década de 1940. O primeiro deles foi o MARK I em 1944, ele foi construído no campus de Harvard.O próximo computador a surgir foi o Colossus em 1946 e logo depois dele veio o considerado pai dos computadores o ENIAC no mesmo ano. Essa foi a geração pioneira

A 2a geração foi marcada pela transição das válvulas para os transistores e na 3ª Geração houve a introdução dos circuítos integrados

A 4a Geração foi onde os computadores pessoais foram introduzidos a sociedade, além de avanços significativos como os microprocessadores e a expansão dos circuítos integrados. Foi também onde as grandes empresas do ramo surgiram como por exemplo a Microsoft(1975) e a Apple(1976)

A 5ª Geração está sendo marcada pelo avanço da Inteligência artificial e do Bitcoin

Breve História da internet:

Antes de surgir a internet, se utilizava o telégrafo para enviar mensagens em código Morse, esse aparelho usava a corrente elétrica para enviar pulsos que eram interpretados por meio do código Morse. Na década de 50, durante a guerra fria surgiu a Darpa (Defense Advanced Research Protect Agency), dentro dessa agência um homem chamado Joseph Carl Robnett Licklider teorizou sobre uma rede galática que pudesse fazer dois computadores se comunicar através dessa rede.

Depois de um tempo surge a Arpanet que estabelece um sistema de comunicação em pacotes, enviados um a um e que permitia múltiplos destinos. Em 1969 foi feito a primeira conexão entre dois computadores, e também a primeira queda de conexão.

O que são os Clients

O lado dos Clients da rede é o lado dos usuários, ou seja, é tudo o que o cliente/ usuário consome através da internet, as imagens, os vídeos, todos os arquivos que são gerados pela conexão com a internet. Os usuários acessam via navegadores ou Browsers (programas criados por empresas para executar os arquivos da internet) ou por aplicações Web (Não tem a necessidade de ser instalado, o meio de comunicação dessa aplicações é a rede) através dos dispositivos móveis (tablet ou smartphone) ou do PC

O que são os Servers

Um servidor é onde estão armazenados, processados e centraliadas as informações que o cliente vai requerer durante a navegação na internet. Isso pode ser feito através de um único computador ou por vários computadores em um data center a depender do tamanho da empresa e da disponibilidade das informações.

Existem alguns tipos de servidores:

  • Arquivos
  • Segurança
  • Streaming
  • E-mail
  • Web

Linguagens de Programação na WEB

A linguagem de programação no ambiente Web é uma linguagem escrita e formal que especifica um cojunto de instruções e regras usadas para gerar programas (Software). No lado do servidor temos as linguagens que farão as requisições no banco de dados dos servidores, são elas: Java, Ruby, Python, JavaScript, C#, etc. No lado do Cliente a linguagem mais usada é o JavaScript (Html não é linguagem de progrmação, é uma linguagem de marcação como o CSV por exemplo)

Tags e seus atributos

Além das tags que são as espinhas dorsais do arquivos Html, há outras tags para serem analisadam e que são muito mportantes:

  • Tag i cria palavras em itálico.
  • Tag strong cria palavras em negrito

  • Tag p cria parágrafos

  • A tag input cria uma caixa de texto
  • A tag img cria um campo onde se tem imagens

Além das tags tem também os atributos para associá-las

  • id para identificador do elemento
  • style para estilizar um determinado elemento fora do CSS
  • class para dar uma classe ao elemento
  • type para identificar o tipo de elemento do campo (como um numero ou texto por exemplo)
  • src para fazer a procura de um arquivo, bastante usado para imagens e vídeos
  • width para setar uma largura ao elemento

Textos

Há tags específicas para textos também. Aqui estão algumas delas:

  • O H1 é o título

  • O H2 é o sub-título

  • O H3 vem abaixo do H2

  • O H4 vem abaixo do H3

  • O H5 vem abaixo do H4
  • O H6 vem abaixo do H5
  • O blockquote é uma tag para citação
  • Mark é uma tag para marcação na página
  • O Sup é uma tag que deixam as letras acima das demais
  • Small é uma tag que deixam as letras pequenas
  • A tag U deixa o texto sublinhado
  • A Tag abbr define uma abreviação ou um acrônimo
  • A tag DEL deixa o texto "deletado" do documento
